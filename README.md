# Real-Debrid API

A simple API interface for pyLoad.

## Getting started

### Install

You may install the pyLoad API with Composer (recommended) or manually.

For composer install :
```
"repositories": [
  {
    "type": "git",
    "url": "https://github.com/ValentinGot/pyload"
  }
],
"require": {
  "php": ">=5.5",
  "vgot/real-debrid": "dev-master"
}
```

### System Requirements

You need **PHP >= 5.5.0**
